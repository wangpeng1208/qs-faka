# 安装方法

* BT后台 添加站点 php版本-静态 ，创建一个mysql数据库。
* 将程序放到 该站点目录下。
* 命令行执行composer安装扩展包，命令：composer install
* 打开命令行 使用 php start.php start -d 命令启动。
* 数据库
    还原sql到数据库中（数据库文件在 public/static/mysql.sql）
    在.env.example 改名为 .env 配置数据库配置
* 设置网站
    反向代理  目标URL : http://127.0.0.1:8786
* 启动服务
    php start.php start -d
* 喝杯咖啡

### 启动时 Address already in use

则可以先强制清理端口占用然后排查错误后重新执行 php webman start -d
. 查看端口占用 lsof -i:8786

. 强制清理端口占用 kill -9 $(lsof -t -i:8786)

### 安装多个时

. /config/thinkcache.php 里的 redis prefix 缓存前缀 改改
. /config/server.php  里的 listen 端口号 改改
.  /config/process.php  里的 listen 端口号 改改

### 命令

. 启动服务 ：  php start.php start -d


. 停止服务 ：   php start.php stop
. 平滑重载 ：   php start.php reload
. 重新启动 ：   php start.php restart

#### 网站其他命令

. 创建自动提现 ：  php start.php auto:cash

. 自动代付任务 ：   php start.php auto:daifu

. 自动解冻订单金额 ：  php start.php  unfreeze:money

. 更多命令详见： php start.php list

# 骑士发卡 版本区别

> 开源版：观望，测试，模板制作，支付接口对接等有二次开发需求用户，亦可以当个人发卡使用

> 年费版：不确定能做多久，新行业试运营 等

> 永久授权版：深耕于发卡行业的又对现况不满足的高手本，对网站性能和安全性有一定的了解和需求

------------------------------------
<span style="color:#00100;font-size: 16px;">

1. 最佳性能：内核为多进程+Epoll+非阻塞IO的webman框架，每个进程能维持上万并发连接，不依赖于php的 php-fpm 。比传统框架性能高出10-100倍，原来10台服务器现在一台搞定。通过调试可以看出接口速度一般在10ms-40ms之间响应完毕。也可以对系统进行二进制打包用于免安装php的环境中。       

2. 页面丝滑：全平台使用(tdesign)vue3 + vite + pinia 开发，页面丝滑流畅。   

3. 极易扩展：系统使用的扩展包都是最新版的安装包，支持现有composer生态；数据库操作使用的thinkorm模型；前端页面vue3 setup写法，封装了部分常用组件。      

4. 尊重习惯：将始终关注用户体验，简化操作流程，降低操作难度，提升系统的便捷性，尊重习惯则现有的企业发卡系统有一定的相似度，免于使用的复杂度，但代码是新开发，仓库可看。        

5. 积极更新：前后端代码皆托管到了开源平台gitee，并保持一直更新中。
